# React CV

updated in November 2020.

## stack
* ReactJS from create-react-app
* Material UI
* npm
* Sass
    * mixins
    * transitions
    * grid
    * flex
* Code Formatter
    * ESlint
    * Prettier

## Install
```
npm install
```
## Run
```
npm start
```
